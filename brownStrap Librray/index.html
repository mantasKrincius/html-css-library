Buttons

Main classes to use for buttons:

  • Colors: btn-accept, btn-cancel, btn-success, btn-danger, btn-warning, btn-info
  • To change size: large-btn or small-btn
  • To make action: btn-action or to add the shadow btn-shadow, simple btn-hover
  • Meanings of buttons:
    • 1. Simple button with shadow
    • 2. Large button with simple hover
    • 3. Small button
    • 4. Disabled button with simple hover
    • 5. Simple button with action, addional it let you "submit" on the form

Main classes to use for liss:

  • Background colors: list-color-success, list-color-danger, list-color-warning
  • Lists box size: small-box, medium-box, large-box
  • Display or not lists styles and decorations: list-style-none and text-decoration-none
  • Gaps between box list-gaps
  • Horizontal lists: horizontal-list
  • List box effects list-effect and list-effect-two
  • List box borders on/off
  • Additional option to use fixed size: fixed-size-small, fixed-size-normal, fixed-size-large
  • Meanings of lists:
    • 1. Simple list with different colors and borders
    • 2. Medium list addinional has link inside and hover
    • 3. Horizontal large list
